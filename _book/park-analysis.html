Analysis of Trip Advisor data relating to South Australian parks

4 Park analysis

4.1 Model

Stars given to a park were analysed by age, gender and origin of reviewer using a Bayesian binomial generalised linear mixed model. The analysis was run using the rstanarm package (Stan Development Team 2016) in R (R Core Team 2018). The model specification was:

cbind(Y, 4 - Y) ~ time * Park + (1 | Age) + (1 | Gender) + (1 | , State) where Y is Stars-1.

4.3 Model predictions
